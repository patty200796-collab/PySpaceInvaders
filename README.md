## 太空侵略者

太空侵略者最初是一款於 1978 年發布的街機遊戲。它是一款 2D 射擊遊戲，也是該類型遊戲的開山之作。

與當時大多數街機遊戲一樣，目標是透過不斷消滅來襲的外星人來獲得最高分。

## 遊戲玩法

許多外星人正在入侵地球，它們緩慢地向你移動，並隨機向你發射雷射。

飛船（即玩家）可以水平移動，躲避雷射攻擊。

太空船也可以向外星人開火，但由於一次只能發射一發子彈，因此需要仔細瞄準。

太空船可以躲在掩體下方，但這些掩體會被敵人的雷射和太空船飛彈逐漸摧毀。

擊中的外星人會被摧毀並獲得分數。飛船也會週期性地出現神秘飛碟，摧毀飛碟可以獲得大量額外分數。
